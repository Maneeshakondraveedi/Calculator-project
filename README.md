# Basic Calculator Web Application

## Level 1 – Task 3

This project is a Basic Calculator Web Application developed using HTML, CSS and JavaScript as part of the Level 1 Internship Task.

The calculator allows users to perform basic arithmetic operations in a simple and user-friendly interface.

## Project Objective

The objective of this project is to design and develop a functional calculator using frontend web technologies. This project helped in understanding HTML structure, CSS styling and JavaScript functionality.

## Features

✔ Addition  
✔ Subtraction  
✔ Multiplication  
✔ Division  
✔ Clear Button  
✔ Delete Button  
✔ Error Handling  
✔ Simple User Interface

## Technologies Used

HTML  
Used to create the structure of the calculator.

CSS  
Used to design and align calculator buttons and layout.

JavaScript  
Used to perform calculations and handle button clicks.

## Project Files

index.html → Structure of calculator

style.css → Design of calculator

script.js → Calculator logic

## How to Run the Project

Step 1:
Download or clone the repository

Step 2:
Open index.html in any browser

The calculator will start working.
